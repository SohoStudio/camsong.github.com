== Description ==

A WordPress code highlight plugin powered by Alex Gorbatchev's SyntaxHighlighter.

Support Language : AppleScript, ActionScript3, Bash/shell, C#, C++, CSS, Delphi, Diff, Groovy, JavaScript, Java, Perl, PHP, Plain Text, Python, Ruby, Sass, Scala, SQL, Visual Basic and XML.


== Usage ==

Use <pre> tag in HTML mode such as
<pre class="brush:php">code here</pre>
