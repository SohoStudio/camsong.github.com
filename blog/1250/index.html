rvm 安装 Ruby 1.9.3时错误： compiler cannot create executables

十二 31st, 2011
发表评论 | Trackback 
rvm install 1.9.3

报错

Installing yaml to /Users/Camel/.rvm/usr
ruby-1.9.3-p0 - #configuring
ERROR: Error running ' ./configure --prefix=/Users/Camel/.rvm/rubies/ruby-1.9.3-p0 --enable-shared --disable-install-doc --with-libyaml-dir=/Users/Camel/.rvm/usr ', please read /Users/Camel/.rvm/log/ruby-1.9.3-p0/configure.log
ERROR: There has been an error while running configure. Halting the installation.

解法

rvm install 1.9.3 --with-gcc=clang

原因未知，安装1.9.2没有问题，估计和xcode有关。
